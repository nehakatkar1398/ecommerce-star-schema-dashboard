# 📊 Ecommerce Star Schema Dashboard
# 🔹 Project Overview
This project demonstrates a well-structured "Star Schema Data Model" combined with an interactive dashboard to analyze ecommerce sales performance.
The objective is to showcase strong data modeling skills along with the ability to generate meaningful business insights.

# 🔹 Data Model (Star Schema)
The data model follows a "Star Schema design1", with a central fact table connected to multiple dimension tables for efficient analysis.

# Structure:
* Fact Table: Orders
* Dimension Tables: Customers, Products, Date

# Relationships:
* Customers → Orders (1:M)
* Products → Orders (1:M)
* Date → Orders (1:M)

A dedicated Date dimension is used to support time intelligence and trend analysis.

# 🔹 Dashboard Features
* KPI Metrics (Total Sales, Orders, Quantity)
* Sales Trend Analysis (Year-wise)
* Customer-level insights
* Clean and minimal design for better readability

# 🔹 Key Insights
* Sales show an upward trend, with highest performance in 2022
* Revenue is concentrated among top customers
* High order completion rate indicates operational efficiency
* Relational data model connects Customers, Orders, and Products for end-to-end analysis
